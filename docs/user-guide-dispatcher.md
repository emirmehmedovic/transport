# Dispatcher User Guide

## Osnovno
- Dispatcher upravlja loadovima i dodjelama vozača/kamiona.

## Glavne aktivnosti
- Kreiranje loadova
- Dodjela loadova vozačima
- Praćenje statusa transporta
- Upload potrebnih dokumenata
- Praćenje alerts-a

## Preporučeni workflow
1. Kreiraj novi load s tačnim pickup/delivery podacima.
2. Dodijeli vozača i kamion.
3. Prati statuse (Assigned → Picked Up → In Transit → Delivered → Completed).
4. Dodaj dokumente i provjeri compliance.

## Schengen 90/180
- Prati upozorenja prije dodjele loadova u Schengen.
