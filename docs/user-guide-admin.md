# Admin User Guide

## Osnovno
- Admin ima puni pristup svim modulima.
- Upravlja korisnicima, postavkama i ključnim operacijama.

## Glavne aktivnosti
- Kreiranje korisnika (dispatchers/drivers)
- Upravljanje vozačima i kamionima
- Kreiranje/assign loadova
- Pregled i rješavanje alertova
- Kreiranje i isplata pay stub-ova

## Preporučeni workflow
1. Kreiraj korisnike i profile vozača/kamiona.
2. Kreiraj load i dodijeli vozača + kamion.
3. Prati status i dokumentaciju.
4. Generiši pay stub-ove i označi isplate.
5. Pregledaj alerts i audit logove.

## Alerts & Compliance
- Alerts se automatski generišu za compliance, maintenance i neplaćene stubs.
- Dismiss koristi se kada je problem riješen.

## Izvještaji
- Koristi standardne reports stranice ili Custom Report Builder.
