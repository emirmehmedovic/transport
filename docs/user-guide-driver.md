# Driver User Guide

## Osnovno
- Driver vidi svoje loadove, status i dokumente.
- Može ažurirati status loada.

## Glavne aktivnosti
- Pregled assigned loadova
- Ažuriranje statusa (pickup, transit, delivery)
- Upload potrebne dokumentacije
- Pregled Schengen statusa

## Preporučeni workflow
1. Otvori load i potvrdi detalje.
2. Prati upute i ažuriraj status po fazama.
3. Upload POD ili druge dokumente.
